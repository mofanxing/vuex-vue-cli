# vuex-vue-cli

## 项目介绍
  这是一个为了方便自己进行vue开发，制定的自己的脚手架工具，
  该项目配置了基础的vuex所需文件和基础代码，也定义了一些目录
  结构，如：api、common、base、components、router、store等

## 项目安装

'''
  npm install -g redlives-cli
'''

## 项目初始化

  '''
    redlives init <templentName> <projectName>
  '''

## 查看帮助信息
  
  '''
    redlives --help
  '''
  
## 查看可用模板列表
  
  '''
    redlives list
  '''
 
 
## 查看版本
  
  '''
    redlives -V
  '''
